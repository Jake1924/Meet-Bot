# Meet-Bot

 Keeping track of google meet/ zoom links for our meetings specially for online class links. This chrome extension makes your life easy! You just have to enter the start and end (optional) time of your meeting in the Custom Tab and this extension would auto join and auto leave the meeting for you! ðŸš€

## Features

- Auto Join Meeting on Schedule Time
- Auto End Meeting after Schedule time
- Automatically disable/mutes Microphone & Camera

### Steps to run ChromeExtension locally

- Clone this repository git clone https://github.com/Jake1924/Meet-Bot.git
- Load your extension on Chrome:
- Go to chrome://extensions/
- Turn on Developer mode
- Click on Load unpacked extension
- Select the clonned folder and upload it
